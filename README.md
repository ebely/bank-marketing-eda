# Bank Marketing EDA

Análisis Exploratorio de Datos (EDA) del dataset Bank Marketing usando Streamlit.

## Autor
Ruth Llalla

## Tecnologías
- Python
- Pandas
- NumPy
- Streamlit
- Matplotlib
- Seaborn

## Objetivo
Analizar factores que influyen en la aceptación de campañas de marketing bancarias.

## Funcionalidades
- Carga de dataset
- Información del dataset
- Estadísticas descriptivas
- Visualización de distribuciones
- Conclusiones del análisis

## Ejecución

Instalar dependencias:

pip install -r requirements.txt

Ejecutar aplicación:

streamlit run app.py
